속성 기능
src 동영상 파일의 경로를 지정
poster 동영상이 화면에 나타나지 않을 때 대신 표시할 그림을 지정
preload 동영상이 백그라운에서 다운로드 되어 재생단추를 클릭할 시 재생
autoplay 동영상 자동 재생
loop 반복재생횟수를 지정
controls 동영상 화면에 컨트롤 막대를 추가
재생/일시정지 단추, 트랙바, 볼륨조절기
- 형태는 브라우저마다 다를 수 있음
type 입력하지 않아도 되지만, 웹브라우저가 파일을 내려받은 뒤에 재생 가능한 파일인지를 확인하게 되면 트래픽 낭비가 생김. 이를 방지 하기 위해 사용함
width 동영상 화면의 너비를 지정
height 동영상 화면의 높이를 지정
